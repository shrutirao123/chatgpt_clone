# AI Chat Assistant (ChatGPT Clone)

A full-stack ChatGPT-like conversational AI web application built using Django and JavaScript.

## Features
- Real-time AI response streaming
- Markdown formatted messages
- Dark and Light mode UI
- Chat history with archive, rename, and delete
- File upload support (images, PDFs, docs)
- Edit and resend message functionality
- Fully responsive design

## Tech Stack
- Python (Django)
- HTML, CSS, JavaScript
- AI APIs
- Markdown (marked.js)

## Note
Database, media files, and vector indexes are excluded from the repository for security and performance reasons.
