# AI Chat Assistant (ChatGPT Clone)

A full-stack ChatGPT-like conversational AI web application built using Django and JavaScript, powered by a **locally hosted offline language model**.

## Key Highlights
- Uses an **offline LLM (no external API dependency)**
- Real-time AI response streaming
- FAISS-based vector search for contextual memory
- Markdown formatted responses (code blocks, lists, headings)
- Dark and Light mode UI
- Chat history with archive, rename, and delete
- File upload support (images, PDFs, documents)
- Fully responsive design

## Tech Stack
- Python (Django)
- HTML, CSS, JavaScript
- Offline Language Model (Local LLM)
- FAISS (Vector Database)
- Markdown (marked.js)

## Note
Database files, uploaded media, and vector indexes are excluded from the repository for security and performance reasons.
